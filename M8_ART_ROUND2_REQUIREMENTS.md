## 二、角色基准约束（所有新图必须遵守）

以下是从 `CHARACTER_STYLE_LOCK.md` 提取的硬性规范，任何新产出的图都必须严格对照 `character_master_idle.png` 来画：

### 固定不变的特征

| 项目 | 规范 |
|------|------|
| 角色 | Q 版桌宠少女 |
| 发型 | 极长的冰蓝/银蓝色头发，垂至下半身 |
| 发饰 | 头顶两个黑色 X 发夹 + 左侧蓝色蝴蝶结 |
| 眼睛 | 大圆眼，饱和蓝色，白色高光 |
| 裙子 | 冰蓝色荷叶边连衣裙，白色蕾丝边，蓝色领口蝴蝶结 |
| 袜子 | 白色过膝袜 |
| 鞋子 | 浅蓝色鞋 |
| 色调 | 裙子保持淡冰蓝色，禁止偏紫/偏粉 |

### 画布与构图规范

| 项目 | 规范 |
|------|------|
| 画布尺寸 | **900 × 900 px**，透明底 PNG |
| 脚底对齐线 | 站立姿态脚底在 **y = 865** |
| 角色高度 | 站立姿态约 **830 px** |
| 全身可见 | 不裁切头发、手、裙摆、腿、袜子、鞋 |

### 禁止事项

- 禁止随机换装（不同图穿不同衣服）
- 禁止换色（裙子颜色必须一致）
- 禁止换鞋/换袜
- 禁止改头身比
- 禁止裁切身体部件

### 参考基准

所有新图请以 `assets/character_masters/character_master_idle.png` 作为第一参考。各表情基准图如下：

| 文件名 | 作为什么场景的参考 |
|--------|------------------|
| `character_master_idle.png` | 标准站姿（所有图的角色基准） |
| `character_master_blink.png` | 闭眼/眨眼 |
| `character_master_smile.png` | 开心表情 |
| `character_master_shy.png` | 害羞表情 |
| `character_master_pout.png` | 鼓脸/生气 |
| `character_master_reach.png` | 伸手动作 |
| `character_master_read.png` | 看书/学习姿态 |

## 四、文件命名与交付格式

### 命名规则

- 分阶段帧使用 `动作名_序号_阶段描述.png`，例如 `pat_head_01_touch.png`
- 道具使用 `prop_物品名.png`，特效使用 `fx_特效名.png`
- 全部小写，下划线分隔

### 文件格式

- **PNG，透明底**
- 角色图：**900 × 900 px**
- 道具零食：64 × 64 px
- 道具正餐：80 × 80 px
- 道具饮品：48 × 64 px
- 特效：48 × 48 px

### 建议的目录结构

```
交付包/
├── animations_m8/
│   ├── hover/
│   │   ├── 000.png  (hover_01_look)
│   │   ├── 001.png  (hover_02_react)
│   │   └── 002.png  (hover_03_settle)
│   ├── pat_head/
│   │   ├── 000.png
│   │   ├── 001.png
│   │   └── 002.png
│   ├── ... (其他动作组同理)
│   └── idle/
│       ├── 003.png  (新增呼吸帧)
│       └── 004.png
└── props_m8/
    ├── prop_cake.png
    ├── prop_candy.png
    └── prop_omurice.png
```

每个动作目录下的帧从 `000.png` 开始编号，代码侧会按编号顺序播放。

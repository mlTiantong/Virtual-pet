# DesktopPet M8 ArtPack v1

This is an art-only package for the desktop pet project. It contains a unified character baseline and a first batch of consistent action resources.

Use `asset-manifest.m8.json`, `animation-manifest.m8.json`, `prop-manifest.m8.json`, and `motion-sequence.m8.json` for integration.
